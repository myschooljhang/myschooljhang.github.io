## 🏫 About

MySchool Jhang is a free educational initiative by **Massab Farooq** serving students in Jhang and beyond. The website connects students to the live learning portal, covering Pakistan's national curriculum for Grades 1–8.

## 🚀 Deployment (GitHub Pages)

This site deploys **automatically** via GitHub Actions on every push to `main`.

### Manual Setup (first time only)

1. Go to your repository: `https://github.com/myschooljhang/myschool`
2. Click **Settings** → **Pages**
3. Under **Source**, select **GitHub Actions**
4. Push any change to `main` — the site deploys automatically

**Your site will be live at:**
```
https://myschooljhang.github.io/myschool
```
